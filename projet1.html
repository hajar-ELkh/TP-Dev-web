Projet 1

Ce Projet académique de module Systeme d’information et bases de données.

À propos du Projet

-Conception en Merise et développement d’un site web pour la gestion administrative d’une clinique. - Utilisation de MySQL pour la gestion sécurisée des données, ainsi que HTML, CSS et JavaScript pour le frontend. - PHP et l’environnement Laragon ont été exploités pour le backend. - Cette plateforme permet une gestion efficace des rendez-vous, dossiers patients, analyses médicales et chambres, offrant une organisation optimisée pour le secteur médical.
